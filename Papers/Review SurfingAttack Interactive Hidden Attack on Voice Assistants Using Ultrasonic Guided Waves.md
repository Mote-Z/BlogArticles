# SurfingAttack: Interactive Hidden Attack on Voice Assistants Using Ultrasonic Guided Waves

> 随着人工智能和自然语言处理的发展，人们可以通过声音来控制自己的设备执行一些简单的命令（比如 Siri、Alexa、Google Assistant 等），根据已有的研究表明，这些语音助手（voice assistant）可以通过听不见的频率注入控制指令，目前为止大部分的研究工作主要集中在通过视线超声换能器来传递单指令控制信号，或者通过扬声器阵列扩展此类攻击的攻击范围。
## 一：本文概要

本文提出了一种新的攻击方式 Surfing Attack，通过语音注入的方式来攻击带有语音助手的智能设备 。与以往通过空气传播的方式相比，Surfing Attack 利用的是超声波在自由固体中发生反射和干涉作用形成导波，使其可以不依赖于空气传播。这种攻击方式，延长了攻击者到智能设备之间的距离，并且实现了多轮交互的效果。作为能控制智能设备执行部分操作的攻击手段，该攻击还可以与其他需要交互的攻击手段进行结合从而造成更大的危害。最后设计了针对不同影响因素的对比试验，分析其对 Surfing Attack的成功率的影响。

前提

1. 假设攻击者知道被攻击设备的唤醒口令，大部分时候是默认激活口令
2. 需要将一个攻击设备放置在与被攻击设备处于同一块板子（比如桌面）的隐蔽地方



## 二：相关原理

### 基带信号

所谓基带信号，是指信息源（发送端）发出的没有经过调制（频谱搬移和变换）的原始电信号，其特点是频率较低，信号频谱从零频附近开始，具有低通形式。根据原始电信号的特征，基带信号可分为数字基带信号和模拟基带信号（相应地，信源也分为数字信源和模拟信源），其由信源决定。直观的比如我们说话的声波就是基带信号。一般而言，基带信号往往不能 作为传输信号，因此必须将基带信号转变为一个相对基带频率而言频率非常高的信号以适合传输。这个信号就叫已调信号，而基带信号叫做调制信号。

### 调制信号

调制信号是由基带信号经过调制得到的高频信号。所谓调制就是按照A信号的特征（比如本文中的攻击指令的信号），改变B信号的某些特征值（比如频率、振幅、相位等），导致B信号的特征值发生有规律的变化（与A信号有关），因此可以根据B信号的变化来还原出A信号，也就是说B信号携带了A信号的相关信息。这就是调制的作用。上述A信号就是基带信号，B信号是已调信号，完成调制的高频信号为调制信号。



### 导波

超声波在板子上传播、反射、干涉所形成的叫导波



## 三：组成及流程

![SurfingAttack 的系统架构](https://blog-1301895608.cos.ap-guangzhou.myqcloud.com/img/20200520095024.png)

- Controller：负责将基带信号（控制指令信号）调制成调制信号发出

- Victim：被动接收周围的信号，加入接收到语音助手的激活信号会唤醒手机

- Attack Device：放置在与被攻击设备同一块板子上，包含了信号处理器、超声波激发器、声音接收器

流程：

1. Controller将控制信号进行调制，通过无线信道发送到 Attack Device 上

2. Attack Device 负责还原控制信号，并转换成超声波发出，超声波在自由固体（桌面）中传播、反射以及干涉作用下形成导波形成振动（需要对频率、类型等进行选择）

3. Victim 接收到振动，并激活语音助手，响应信息会回传到 Attack Device中，并传回Attacker处解码

关于此处，Victim的响应是以什么形式回传到 Attack Device的，我并没有在论文中看到，这是我觉得奇怪的点，如果是声音传播，那必然会引起受害者的注意，但是如果是以同样导波这种形式的话，手机可以发出超声波吗？这是比较奇怪的地方。

## 四：攻击场景

![](https://blog-1301895608.cos.ap-guangzhou.myqcloud.com/img/20200520104715.png)

人们常常会在工作时把手机等设备放置在桌子的一旁，主要精力集中在电脑上，通过 Surfing Attack ，攻击者可以隐蔽的唤醒受害者的设备，并注入语音命令，比如唤醒设备、调低音量、拨打电话、阅读验证码等攻击行为。随着短信验证码成为主流的身份验证方式，一旦获取到短信验证码，受害者的设备所绑定的账号都面临着安全问题，并且，该攻击还可以与其他需要交互配合的攻击结合使用，达到更为显著的攻击效果。





## 五：Surfing Attack 的关键要素

**成功实施 Surfing Attack 的三个必要条件**

- 桌子上的超声波必须能够到达设备外壳中包含的麦克风设备
	- 与以往通过空气传播不同的是，声波在固体材料中传播具有声扩散的现象，在此过程中，声波通过材料时会被分离为其组成频率
	- 需要实现四个目标：（1）低分离度、（2）弱衰减性、（3）容易激活、（4）攻击信号的高可达性
	- 为了实现上述目标，需要考虑：（1）信号波形、（2）Lamb wave 模式 、（3）超声波信号源
	- 根据现有的研究，窄带输入信号对于限制大厚度板中的波扩散最有效，因此，在 `SurfingAttack`中选择窄带窗口调制信号来携带攻击指令以达到最小扩散
	- 选择低于`higher order Lamb wave modes`截止频率的  `A0 wave mode `来创建超声波指令
	- 基于能源效率以及覆盖性的考虑选择 `circular piezoelectric disc`来产生信号

- 即使麦克风可能未与传输介质直接接触，电波仍应能够利用桌面上的麦克风设备的非线性效应来发起听不见的命令注入攻击

	- 通过实验验证手机的麦克风等语音捕获硬件的非线性效应可以由在桌子上传播的超声波引导波触发。

- Attacker 可以通过植入的设备接收 Victim 的响应,而不会引起受害用户的注意

	- 通过 `Sound pressure level (SPL)`来作为衡量听觉引起警觉的一个评判标准
	- 实验表明将音量调得足够低可以使得用户在适当距离内无法察觉语音响应，而隐藏的窃听设备仍然可以捕获声音
	- 可以通过隐蔽放置多个设备来捕获来自设备扬声器的微弱声音响应
	- 可以根据背景音量适当调整攻击时 Victim 的响应音量



**超声波信号产生**

1. 使用窄带窗口来调制带有语音命令的调制信号，在激发 Lamb wave之前需要做预处理来保证（1）适当的频率带宽、（2）适当的调制信号

2. 为了激活语音助手（Google Assistent、Siri等），基带信号会将唤醒指令放在攻击指令前面，并且在两者之间插入几秒的间隔（避免唤醒时的震动行为导致的影响），唤醒信号以及声音可以根据前人的研究结果来生成$^{[33]}$，攻击指令可以简单的使用TTS来产生



## 六：实验探究

**实验验证**

通过对 17款带有语音助手的主流手机设备进行实验，验证了 SurfingAttack 的可行性、量化了成功攻击所需要的条件（相关参数）、测试了攻击命令的识别度

**实验准备**

- 实验在40dB SPL左右的背景噪声环境中进行
- 如何衡量攻击的泛化能力，即攻击是否具有普适性，而不是单纯针对某类硬件或者特定型号

**实验结果**

- 攻击可行性：成功攻击 17款设备中的15款，攻击信号频率 $f_c$ 是成功的主要因素，并且验证了即使在 in-air 信道被阻断的情况下，SurfingAttack 仍然可以攻击成功

![实验结果](https://blog-1301895608.cos.ap-guangzhou.myqcloud.com/img/20200520114238.png)

- 失败的原因分析：通过对比发现，并不是系统原因，而是手机的架构以及使用的材料
- 攻击隐蔽性分析：根据不同级别的音量来衡量其隐蔽性，在拨打电话实验中发现，被叫方的响应内容在音量级别为 1~5 的环境噪声中被掩盖  攻击隐蔽性的[实验视频](https://youtu.be/zgr-oM2YJHs)



**环境影响分析**

- 背景噪声分析：由于超声波导波的能量传递形式，SurfingAttack 攻击受环境影响较小
- 对话环境中的影响：即使在最大对话音量的环境中，识别率也是可以接受的，短指令可能受对话环境的干扰比较小，但是对一些长指令来说，可能会受对话影响

![距离与识别率](https://blog-1301895608.cos.ap-guangzhou.myqcloud.com/img/20200520122438.png)

- 方向性影响分析：超声波信号在空气中传播是具有方向性的，结果表明 SurfingAttack是全方位的，可以有效针对桌面上任意位置和方向的任何设备

- 攻击距离的影响分析：主要是由于高能转换器的转换效率以及超声波随距离的衰减两方面影响，实验结果表明 SurfingAttack 可以远距离较高成功率的攻击语音助手的设备
- 桌面材料的影响分析：实验结果表明 SurfingAttack 严重依赖于桌子的材料，因为这会影响导波的产生，传播以及与设备的机械耦合等特性，并且，在桌子材质与测试设备的材质相匹配时，会大大提高攻击的成功率
- 桌面厚度的影响分析：实验结果表明桌面的厚度并不会显著影响攻击的效果



## 七：总结

​	这类攻击的攻击方式比较新颖，关注到了与传统攻击途径（沿空气传播）所不同的（超声导波在固体传播），这类攻击的攻击路径是通过桌子作为介质的，沿桌子传播的形式不受桌面环境的影响，比如杂乱的堆放影响声音的传播，并且可以同时对多个设备进行攻击，但是这种攻击方法受桌子的材料限制比较大，并且在安卓某次更新以后（用户可以自定义唤醒口令），因此需要知道受害设备的唤醒口令，具有较明显的局限性，并且想要进行Surfing Attack需要利用一个小物件攻击设备来做中转，这就使得这项攻击技术的利用难度大大增加了，不过不失为一个好玩的攻击思路。

​	在其中有几个地方值得注意，一个是需要对手机搜集振动信息比较了解，第二个就是，手机发出的响应怎么收集，这两个地方在论文中并没有做解释。

​	从实验验证的结果来看，Surfing Attack这种沿固体传播的攻击形式，较为隐蔽，只要前置条件都满足，可以达到很好的效果。
